## 失败模式归档

### 模式 1：Session 过期
**症状：** 填完表单提交，返回登录页。
**根因：** Session cookie 过期或被 CSRF token 检查拦截。
**预防：** 提交前检查 session 状态，必要时重新登录。

### 模式 2：静默拒绝
**症状：** 提交显示成功，但内容从未上线。
**根因：** 邮箱域名被标记 / IP 被标记 / 内容触发后端审核。
**预防：** 用干净 Gmail + 检查 live URL。

### 模式 3：字段映射错误
**症状：** 提交成功但信息显示不正确。
**根因：** 平台改版，选择器失效。
**预防：** 每次提交前截图确认页面结构。

### 模式 4：验证码一次性浪费
**症状：** 验证码过了但提交失败（漏填必填字段）。
**根因：** 铁律 10 违反。
**预防：** 先填完所有字段，只剩验证码才叫人。
